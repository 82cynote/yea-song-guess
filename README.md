# 🎵 抽歌播放器 (Song Guessing Player)

這是一個專為猜歌遊戲設計的網頁播放器。只需要貼上 YouTube 播放清單，就能自動載入題庫，並具備隨機跳轉、隱藏歌名與動態計分板等功能，讓派對或活動的猜歌環節更順暢。

## 一、 遊戲前置設定
1. **準備歌單**：請前往 YouTube 複製你要播放的清單網址。
   * *注意：請確保網址帶有 `list=PL...` 開頭。若是 `list=RD...` 開頭代表是 YouTube 自動生成的無盡合輯，可能會抓取到清單外的關聯歌曲。另外，播放清單權限必須是「公開」或「不公開」。*
2. **載入資料**：將網址貼入「⚙️ 遊戲設定」中的「YouTube 播放清單網址」欄位，點擊「載入歌單資料」。狀態列顯示綠字即代表成功。
3. **設定隊伍**：在設定中選擇參賽隊伍組數（1-8 組），並填入隊伍名稱。系統會自動記憶這些設定，下次打開不用重填。

## 二、 遊戲進行方式
* **隨機起點**：預設開啟「隨機播放起點 (0% - 90%)」。每次抽歌時，系統會自動跳過前奏，從歌曲中段隨機開始播放，增加猜歌難度。
* **抽歌與播放**：點擊「✔️ 抽一首」，系統會從題庫中隨機挑選一首歌並自動播放，此時答案處於隱藏狀態。
* **揭曉與計分**：有人猜出歌曲後，點擊「👀 公佈答案」顯示歌名。公佈後即可在下方計分板針對得分隊伍點擊 `+1`。若不小心按錯，隨時可以點擊 `-1` 扣回分數（最低為 0 分）。
* **暫停與繼續**：遊戲過程中可隨時點擊「⏯ 暫停」來控制播放狀態，或是拖曳下方的進度條跳轉聆聽不同段落。
